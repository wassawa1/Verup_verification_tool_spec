# テストケース一覧

このディレクトリには Verilog テストケースが含まれています。

## テストケース

### 1. counter.v
- **説明**: 8ビットインクリメンタルカウンター
- **機能**: リセット機能付きのシンプルなカウンター
- **使用信号**: clk, rst, count[7:0]
- **テスト**: リセット、インクリメント動作

### 2. adder.v
- **説明**: 4ビット加算器（キャリー付き）
- **機能**: a + b + cin = {cout, sum}
- **使用信号**: a[3:0], b[3:0], cin, sum[3:0], cout
- **テスト**: すべての組み合わせ (16×16×2 = 512パターン)

### 3. mux.v
- **説明**: 8対1マルチプレクサ
- **機能**: 8ビット入力から選択信号で1ビット出力
- **使用信号**: in[7:0], sel[2:0], out
- **テスト**: すべての入力パターンと選択線 (256×8 = 2048パターン)

### 4. mem.v
- **説明**: 8×8ビットレジスタメモリ
- **機能**: アドレス指定の読み書き
- **使用信号**: clk, we, addr[2:0], din[7:0], dout[7:0]
- **テスト**: 書き込み・読み出し動作検証

### 5. fsm.v
- **説明**: 信号機 FSM（有限状態機械）
- **機能**: 赤 → 緑 → 黄 → 赤 の状態遷移
- **使用信号**: clk, rst, light[2:0]
- **テスト**: 状態遷移タイミング検証

## コンパイルと実行

```bash
# コンパイル
iverilog -o <name>.vvp <name>.v

# 実行
vvp <name>.vvp
```

## VCD ファイル生成

各テストケースは自動的に VCD（Value Change Dump）ファイルを生成します：
- `counter.vcd` - 小規模（~1KB）
- `adder.vcd` - 中規模（~30KB）
- `mux.vcd` - 大規模（~70KB）
- `mem.vcd` - 小規模（~2KB）
- `fsm.vcd` - 小規模（~5KB）

これらは波形比較テストに使用されます。
